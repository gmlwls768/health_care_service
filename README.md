## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/gmlwls768/health_care_service.git
    ```
2. Install the necessary dependencies:
    ```bash
    npm install
    ```
3. Start the application:
    ```bash
    npm start
    ```
